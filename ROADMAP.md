# Encralink Roadmap

## Phase 1: User Adoption
- Drive user growth and generate fees.
- Expand messaging and document sharing features.

## Phase 2: Ecosystem Expansion
- Add support for Ethereum, Base, and other chains.
- Cross‑chain messaging and storage integration.

## Phase 3: Encrypted Mailing System
- Launch a decentralized mailing system similar to email.
- Enable secure communication across wallets.

## Phase 4: Tokenomics Enhancements
- Weekly token locking after buybacks.
- Strengthen fee capture and community trust.

## Phase 5: Governance
- Introduce DAO governance features for community decision‑making.
- Voting and proposal system integrated with the app.

## Phase 6: Accessibility
- Mobile app release for iOS and Android.
- Improved UX for onboarding and group chats.

## Phase 7: Analytics
- Advanced dashboard for transaction history, fee capture, and usage metrics.
